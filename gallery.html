About These Images

These images display different features and functionalities of the Twitter Clone Application.

A typical user-flow of the web app is that first, the user is presented with a home page, that takes the appearance of a scrolling feed. From that screen, a user can click a post and view the full post, as well as click the new post button in the navigation bar. Check out this Youtube video to watch a walk-through of the app, or check out the Github repository to view the code behind the application.
